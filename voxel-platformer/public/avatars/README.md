# Dialogue Avatars

These small PNG files back the built-in dialogue avatar keys:

- `keeper` -> `keeper.png`
- `player` -> `player.png`
- `sundial` -> `sundial.png`
- `book` -> `book.png`
- `npc` -> `npc.png`

Replace any file with another optimized PNG of the same name to update that
portrait without changing level scripts. Scripts can also pass an explicit
image path such as `/avatars/merchant.png`.
